# 数据字典

## sales.csv 字段说明

| 字段名 | 含义 |
|---|---|
| order_id | 订单编号 |
| date | 订单日期 |
| region | 销售区域 |
| city | 城市 |
| category | 产品品类 |
| product | 产品名称 |
| customer_type | 客户类型 |
| sales_amount | 销售额 |
| quantity | 销售数量 |
| discount | 折扣比例 |
| profit | 利润 |
| channel | 销售渠道 |
| status | 订单状态 |

## 核心指标说明

总销售额：统计非退款订单的销售额总和。

总利润：统计所有订单的利润总和，退款订单可能产生负利润。

订单数量：当前筛选条件下的订单总数。

利润率：总利润 / 总销售额。

退款率：退款订单数量 / 总订单数量。

## 数据质量检查

系统会统计总数据行数、缺失关键字段行数、重复订单号数量、已完成订单数量、退款订单数量和待处理订单数量。
